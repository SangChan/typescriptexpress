# TYPESCRIPTEXPRESS

- A server side application that built by express with typescript on NodeJS.
- NodeJS
    - TypeScript
    - Express
    - Swagger

# How to run

## run dev mode

<pre><code> npm run dev </code></pre>

## generate swagger docs

<pre><code> npm run prestart </code></pre>
